# Crime-Reporting-System-

Crime Reporting System is the mini project for reporting complaints of any kind of crime. This project mainly helps to reporting complaints as soon as possible. In this project for front-end HTML, CSS, Bootstrap and for back-end PHP use. The users of this project are complainer, inspectors. The complainer can easily register, log in, complain with necessary details about the crime and logout. The inspector can log in view complaints handle the complaints and takes steps against the crime. For handling the full project details there will be an admin panel. Admin can log in and update delete. The inspector will change the status according to situations.
